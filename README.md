# indexofficial-leyu-notes

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面，内容涉及日常笔记、工具页面、文档索引等。每个页面均为独立文件，不依赖特定后端服务。

仓库名称中的 `indexofficial-leyu-notes` 仅为标识用途，不代表任何具体组织或网站。

## 目录说明

```
.
├── README.md          # 本文件
├── page1.html         # 示例页面 1（实际页面以仓库内文件为准）
├── page2.html         # 示例页面 2
└── ...                # 其他 HTML 文件
```

仓库根目录下存放所有 HTML 文件，文件名即页面标题或功能简述。

## 页面归档说明

- 所有页面均为静态 HTML 文件，可直接在浏览器中打开使用。
- 页面内容可能包含个人笔记、工具界面、信息汇总等，不针对任何特定网站或服务。
- 部分页面可能引用外部资源（如 CSS/JS 库），请确保网络通畅以正常加载。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/indexofficial-leyu-notes.git
   ```
2. 直接双击任意 `.html` 文件即可在浏览器中查看。
3. 也可通过 GitHub Pages 或其他静态托管服务部署整个仓库。
